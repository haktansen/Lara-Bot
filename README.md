🤖 Lara-Bot: Çok Dilli Oyun Takip Otomasyonu
Lara-Bot, Epic Games üzerindeki ücretsiz oyun fırsatlarını anlık olarak takip eden ve kullanıcılara tercih ettikleri dilde bildirim gönderen gelişmiş bir Telegram botudur.

✨ Öne Çıkan Özellikler
7/24 Kesintisiz Çalışma: GitHub Actions altyapısı sayesinde herhangi bir sunucu maliyeti olmadan her 30 dakikada bir otomatik kontrol sağlar.

Çok Dilli Destek: deep-translator entegrasyonu ile bildirimleri kullanıcıların dil tercihlerine göre anlık olarak çevirir.

Gelişmiş Veritabanı: JSON tabanlı hafif bir veritabanı yapısı ile kullanıcı tercihlerini ve istatistikleri (oyun sayıları, tasarruf miktarı vb.) yönetir.

Admin Paneli: Duyuru yönetimi ve sistem istatistiklerini takip etmek için özel yönetim komutları içerir.

🛠️ Kullanılan Teknolojiler
Dil: Python

Kütüphaneler: python-telegram-bot (JobQueue destekli), httpx, deep-translator.

Otomasyon: GitHub Actions (CI/CD).
